# Project Layout

- `main.py` - entry point
- `interactive_web.py` - FastAPI backend and session logic
- `web/index.html` - frontend UI (HTML/CSS/JS)
- `src/interactive/sam_service.py` - SAM runtime and embedding cache
- `src/interactive/runtime.py` - prefetch queue and async save/autosave workers
- `src/interactive/exporter.py` - COCO/YOLO export utilities
